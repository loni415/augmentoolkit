# 海政宣传文化中心文化服务站简介  

海政宣传文化中心文化服务站是由原海政文工团、海政电视 艺术中心、海政创作室等单位调整组建,其前身有着辉煌的历史, 曾创作了一大批具有鲜明时代特点的经典佳作。歌曲《人民海军 向前进》《我爱这蓝色的海洋》《太阳最红毛主席最亲》《泉水 叮咚响》《军港之夜》《爱的奉献》《党啊我想对你说》等在军 地广为传唱。舞蹈《刀锋》《八女投江》《深潜深潜》,歌剧《红 珊瑚》,音乐剧《赤道雨》,电视剧《火蓝刀锋》,电影《红海 行动》等深受观众喜爱。  

文化服务站自组建以来，坚持姓军为战、为兵服务、为战斗力服务，推出的《信仰的味道》《出征战歌》等一系列兵味、海味、战味浓郁的音乐作品，深受官兵喜爱，配合抗击新冠疫情，倾情创作了《跟我上》《前行者》《背影》等多首抗疫MV作品，在全网发布，广受好评。近年来还圆满完成人民海军成立70周年主题文艺晚会、海军机关新春团拜会、“万里海疆行”慰问演出等任务，为提高部队战斗力、鼓舞军心士气做出了积极贡献。  

# 海政宣传文化中心军乐队简介  

海政宣传文化中心军乐队(原海军军乐团)作为海军唯一司礼乐团,先后完成国庆阅兵、海上阅兵等重大庆典演奏任务,以及重要外事活动司礼演奏任务,出访60多个国家和地区,多次受邀参加国际军乐节、海军节、管乐艺术节,被誉为“展示海军形象的靓丽名片”。军乐队专业特色鲜明,致力探索丰富交响管乐、现代行进乐、轻音乐、爵士乐等多种表演形式,创作改编《巨浪》、《海之颂》、《红蓝交响》、《红旗颂》等大型交响管乐作品,录制发行《海军礼仪用曲》、《大海雄风》等音乐专辑,创作编排《太平洋的梦》、《走进轮机舱》、《与祖国同行》等军乐行进表演作品,影响广泛,反响良好。  

# 招考岗位要求  

# 一、四级美术创作员岗位（共2 个）  

(一)国画  

1．招考数量：1 名。  

2.招考条件：①年龄不超过30周岁；②限男性；③高校毕 业生或社会人才；④具有重点专业类艺术院校全日制研究生及 以上学历、学位；⑤所学专业一本科：中国画，研究生：美术 （中国画方向）；⑥个人原创作品获省部级以上美术展览奖项 或入选全国、全军美展；⑦中国美术家协会会员优先。  

3. 从事工作：从事军旅题材美术作品创作。  

4.面试考核：①政治立场、求职动机、适岗能力等。采取 现场答辩方式组织，每人5-10 分钟。②基本功测试。采取现场 按指定题目创作方式，时间约30\-60 分钟。③专业能力现场测 试，采取现场抽题创作方式，时间约180 分钟。④自备绘画工 具，考场提供画纸。自带作品一幅并作简要阐释，携获奖或参 展证书原件。  

# (二)油画  

1．招考数量：1 名。  

2.招考条件：①年龄不超过30 周岁；②限男性；③高校毕  

业生或社会人才; ④具有重点专业类艺术院校全日制研究生及以上学历、学位; ⑤所学专业一本科: 油画, 研究生: 美术 (油画方向); ⑥个人原创作品获省部级以上美术展览奖项或入选全国、全军美展。⑦中国美术家协会会员优先。  

3.从事工作：主要从事军旅题材美术作品创作。  

4.面试考核：①求职动机、适岗能力等。采取面试答辩方式 组织，每人5\-10 分钟。②基本功测试，时间约30 分钟③专业能 力现场测试，时间约180 分钟。④自备绘画工具，考场提供画纸。 自带作品一幅并作简要阐释，携获奖或参展证书原件。  

# 二、三级作曲岗位  

1．招考数量：1 名。  

2.招考条件：①性别不限，45周岁以下；②高校毕业生或 社会人才；③全日制本科以上学历、学士以上学位；④所学专业 ——本科，音乐与舞蹈学类；研究生，音乐（专业学位）；⑤有 中级专业技术资格。  

3.从事工作：大型军乐交响乐的创作、编曲。  

4.面试考核：①政治立场、求职动机、适岗能力等。采取现 场答辩方式组织，每人15\-20 分钟；②专业能力。面试前一天应 用人员现场进行30 小节管乐队作品创作（现场提供音乐创作主 题、乐队编制表、总谱纸等），时限3.5 小时。面试当天，现场 向专家评委解读总谱。  

# 三、四级导演岗位  

1．招考数量：1 名。  

2.招考条件：①性别不限，35周岁以下；②高校毕业生或 社会人才；③全日制本科以上学历、学士以上学位；④所学专业 ——本科，音乐表演；研究生，音乐（专业学位）；⑤熟悉西洋 管乐教学工作；⑥需提交三套行进表演方案(传统军乐队列表演、 现代行进管乐、行进打击乐等形式各一套)的3D动画视频与队 形图谱；⑦因仪式与行进乐队指挥工作性质需要,男性身高185cm 以上，女性身高170cm以上。  

3. 从事工作：行进表演导演、音乐编导。  

4.面试考核：①政治立场、求职动机、适岗能力等。采取现 场答辩方式组织，每人5\-10 分钟；②专业能力。面试前一天应 用人员现场创编制作（现场提供音乐素材等，自行准备所需设 备），时限3.5 小时。面试当天向专家评委解读方案并回答相关 专业问题。③西洋乐器，铜管、水管、打击乐任选其一现场演奏。  

以上岗位入围比例均为1：3。  

# 报考人员须知  

# 一、招考对象  

符合招考岗位要求的高等院校应届毕业生或社会人才。  

# 二、基本条件  

1. 热爱祖国，拥护党的领导，遵纪守法；  

2.热爱军队文化艺术事业，志愿献身国防建设；  

3. 品行端正，身心健康。  

# 三、报名要求  

# （一）报名时间  

报考人员请关注中国人民解放军军队人才网 （http://www.81rc.mil.cn），报名时间自军队人才网发布公告 之日起，至2023年10月14日18:00止，报考人员须在报名时 限内将资格审核材料按规定发送到报考岗位指定邮箱。

身份证，如身份证丢失，提供临时身份证或护照等证明本人 身份的资料。  

# 2.学历、学位材料  

①本科、研究生毕业证，学士、硕士学位证。尚未毕业的应 届生应出具学生证和所在院校就业指导部门提供的在读证明，以 及《教育部学籍在线验证报告》。  

②国内院校毕业人员需提供《教育部学历证书电子注册备案 表》《教育部学籍在线验证报告》。  

③留学回国人员需提供教育部留学服务中心出具的学历认 证书和我国驻外使（领）馆出具的留学回国人员证明。  

# 3.获奖证书  

如有在国内、国际本专业权威比赛中获奖经历，提供获奖证 书等证明材料。
